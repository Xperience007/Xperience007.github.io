My Latest Project: Barred From Freedom

❮ ❯

This is a video game that was made for my senior project. It's a rougelike type game where you get items after you beat a wave of enemies and enventually you fight a final boss. I wanted to make a rougelike game since that's one of my favorite genres and I was definitely inspired by the Risk of Rain series. The game was made in the Unity Engine and all of the core scripts were created using C#. I also used Blender to create both the player and enemy models. This game was completed and delivered within one semester.

Pokémon Image Classifier Project

This is an image classifier that was made to classify the original 151 Pokémon. I created a Convolutional Neural Network to train on a dataset of images and my model has an accuracy of 93%. Python and PyTorch were used in this project. Even though this machine learning model was trained on a specific dataset, it can recognize Pokémon from drawings as seen by the example with Psyduck. Dataset: https://www.kaggle.com/datasets/echometerhhwl/pokemon-gen-1-38914
